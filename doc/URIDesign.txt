fqdn
  Top page.
  If has no session, redirect to login / sign up page.
  If has session, redirect to /rooms.
fqdn/rooms
  Room list.
  User can create new room from this page.
  Room owner can delete one's room from this page.
fqdn/room/xxxxxxxxxxxxxxxx
  Each room, in other words, message list.
  Max room ID is 9999999999999999. No zero padding.
  User can send message from this page.
